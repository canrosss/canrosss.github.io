====== Personalización de GnomeShell ======
Created domingo 23 diciembre 2018

Configuración de la fuente del tema de GNOME Shell
Cada tema tiene su archivo de configuración en CSS el cual puede permitir personalizar la fuente, tamaño y opciones 
Para el caso de VERTEX, como lo tengo instalado en /usr/share/themes/Vertex tengo que entrar al archivo de configuración

cd /usr/share/themes/Vertex
vim gnome-shell/gnome-shell.css

Y localizar las lineas, font-family y font-size
